# Новое русское вино

Сайт магазина авторского вина "Новое русское вино".

## Excel файл
Ваш **Excel** (.xlsx) файл должен выглядит примерно так
|Категория|Название|Сорт|Цена|Картинка|Акция|
|---|----|----|----|----|----|
|Напитки|Чача| |299|Chacha.png|Выгодное предложение|
|Красные вина|Саперави|Саперави|599|Saperavi.png| 

Данные могут быть изменены, но столбец *Категории* **обязателен**.

## Установка
- Скачайте код.
- Добавьте в папку с репозиторием ваш **Excel** файл.
- Установите все нужные библиотеки командой
  ```
  py -m pip install -r requirements.txt
  ```
## Запуск

- Запустите сайт командой
  ```
  python3 main.py -w Example.xlsx
  ```
  Где **Example** это название вашего файла. (По дефолту wine.xlsx)
- Перейдите на сайт по адресу [http://127.0.0.1:8000](http://127.0.0.1:8000).
